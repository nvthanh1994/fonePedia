API

GET /api/phones   - Lists all phones
POST /api/phones  - Create a new phone
GET /api/phones/:id - Returns the data for that specific phone with the id of :id
PUT /api/phones/:id - Updates the information for a specific phone
DELETE /api/post/:id - Deletes the specific post

GET /api/brand - List all brand
GET /api/brand/:id - List all phones by brand :id
